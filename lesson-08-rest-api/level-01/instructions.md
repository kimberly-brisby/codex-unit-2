# Level 01 — What is REST API

## Objective

Introduce the concept of REST APIs and how clients and servers exchange data.

## Benefits

- **Learn concept**: Understand client-server communication.
- **Foundation**: Prepares for using `fetch` in later levels.

## Complete these tasks

- **Select a target element**: save `document.getElementById('result')` to a variable.
- **Set innerText**: set the element's `innerText` to a short explanation mentioning REST APIs or client-server data exchange.

## Hints

- What does a client send and what does a server return?
- Keep the explanation one or two sentences.

## More information

- A REST API is an HTTP-based interface where clients make requests (GET/POST) and servers return responses (often JSON).
- Use HTTP methods semantically: `GET` to read, `POST` to create or send data.

## Usage tips

- Keep text concise; tests expect the explanation in the `result` element.

## Example

```html
<p id="result"></p>
```

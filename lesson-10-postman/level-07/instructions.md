## Level 07 — API POST practice

## Objective

- Choose an API and make a POST request with an appropriate JSON payload.

## Benefits

- Practice constructing payloads and interpreting API responses.
- Reinforce reading schema details in API docs to avoid validation errors.

## Complete these tasks

- Pick an API from the instructor list and find a suitable POST endpoint.
- Read the API docs.
- Make the POST request with data and inspect the response.
- Use `script.js` to `console.log` one fact learned from the documentation.

## Hints

- Check the docs for required/optional fields and example payloads.

## More information

- Some test APIs return an echoed version of your payload for verification; others return a created resource with an ID.

## Usage tips

- Start with minimal required fields to reduce validation errors.

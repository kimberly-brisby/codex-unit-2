## Level 14 — Mini project: Build a small form handler

## Objective

Combine the concepts in this lesson to build a form and a handler.

## Benefits

- **Integration**: Practice combining selection, events, and form APIs.

## Complete these tasks

- Add two inputs with `name` attributes and a submit button.
- Attach `handleSubmit` to `form.onsubmit` and call `preventDefault()`.
- Read values from `form.elements` into separate variables.
- Use an `if` condition to decide when to call `form.reset()`.
- Select the `output` tag and update `innerText` or `innerHTML`.

## Hints

- If using radio buttons, provide `value` attributes also.
- Convert range values with `Number()` when used.

## More information

- Keep the handler small: read values, decide, update output, optionally reset.

## Usage tips

- Test small bits in the console before wiring them together.

# Level 14 — Mini-project

## Objective

Build a small form handler that prevents default submission, reads values via `form.elements`, validates fields, displays a summary in the DOM, and resets the form on success.

## Benefits

- Combines selection, submission handling, value reading, and UI updates.

## Complete these tasks

- Prevent default navigation in the handler.
- Read values for `name`, `message`, `volume`, selected radio `color`, and checkbox `subscribe` via `form.elements`.
- Validate one or two fields, display a summary in `#summary`, and call `form.reset()` on success.

## Hints

- Convert numeric values with `Number()` and verify required fields are not empty.

## More information

- Keep logic simple: show how to gather values and present them in the DOM.

## Usage tips

- Update `#summary` with `innerText` or `innerHTML` for quick feedback.
